# Koloroj.

## Bazaj koloroj.

ruĝa, blua, verda, flava.

la pomo estas ruĝa.
la ĉielo estas blua.
la herbo estas verda.
la suno estas flava.
la tomato estas ruĝa kaj ronda.
la maro estas blua kaj profunda.

## Lumaj kaj malhelaj.

nigra, blanka, griza.

la nokto estas nigra.
la neĝo estas blanka.
la ĉielo estas griza.
la kato estas nigra kun blankaj piedoj.
la elefanto estas griza kaj granda.

## Pliaj koloroj.

bruna, rozkolora, oranĝkolora, violkolora.

la urso estas bruna.
la floro estas rozkolora.
la oranĝo estas oranĝkolora.
la siringo estas violkolora.
la ĉokolado estas bruna.
la robo estas rozkolora kaj bela.
