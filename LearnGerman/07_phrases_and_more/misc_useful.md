# Nützliches.

## Um Hilfe bitten.

Hilfe.
ich brauche Hilfe.
können Sie mir helfen.
kannst du mir helfen.

Hilfe, ich habe ein Problem.
ich brauche Hilfe, ich habe mich verlaufen.
können Sie mir bitte helfen, das zu tragen.
kannst du mir helfen, diese Aufgabe zu lösen.

## Notfälle.

ich bin krank.
mir geht es nicht gut.
ich brauche einen Arzt.
rufen Sie einen Krankenwagen.
wo ist das nächste Krankenhaus.

ich bin krank und kann heute nicht kommen.
mir geht es nicht gut, ich muss mich hinlegen.
ich brauche einen Arzt, es ist dringend.
rufen Sie bitte einen Krankenwagen, schnell.
wo ist das nächste Krankenhaus von hier.

## Verständigung.

sprechen Sie Englisch.
ich spreche nur ein bisschen Deutsch.
können Sie bitte langsamer sprechen.
können Sie das bitte wiederholen.
wie sagt man das auf Deutsch.
wie schreibt man das.

sprechen Sie Englisch, mein Deutsch ist noch nicht gut.
ich spreche nur ein bisschen Deutsch, bitte haben Sie Geduld.
können Sie bitte langsamer sprechen, ich komme nicht mit.
können Sie das bitte wiederholen, ich habe es nicht verstanden.
wie sagt man dieses Wort auf Deutsch.
wie schreibt man Ihren Namen, buchstabieren Sie bitte.
