# Learn German — A Guided Audio Tour

This is an **input-only** book for passive language exposure. It's designed to be read aloud by a text-to-speech engine while you listen — while commuting, doing dishes, walking, or sitting down and reading along.

German has excellent TTS support on all major engines. Pick any one and start listening.

## How to Use

1. Open any `.md` file in this folder.
2. Feed it to a German TTS engine.
3. Listen. Let the sounds wash over you.
4. If you want, open the same file and read along with your eyes while the TTS speaks.
5. Repeat. Come back later. There's no test.

## The Tour (in order)

| Volume | What You'll Hear |
|---|---|
| 00_foundation | The alphabet, umlauts, ß, key sounds |
| 01_numbers | Counting from zero to one hundred, ordinals |
| 02_around_you | Objects, places, colors, adjectives |
| 03_actions | Common verbs, directions |
| 04_social | Greetings, polite expressions, introductions |
| 05_time_and_weather | Days, months, telling time, weather |
| 06_food_and_shopping | Food, drink, shopping phrases, restaurant talk |
| 07_phrases_and_more | Questions, answers, useful miscellany |
| lang_specific | German-specific features: articles and genders, number structure |

Start at 00_foundation and 01_numbers first. After that, any order works.

## What This Is Not

- Not a grammar book
- Not a vocabulary drill system
- Not an exercise workbook
- Not a "master German in 30 days" program

This is a **tour**. You're passing through the language. Hear it. Get used to its rhythm. If some things stick, great. If not, that's fine too.
